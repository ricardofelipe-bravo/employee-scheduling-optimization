# Figures

This folder contains visualizations generated from the optimization results of the workforce scheduling model.

## Description

The figures illustrate key aspects of the optimization outcomes, including demand coverage, staffing allocation across time slots, and performance comparisons between the implemented solution methods.

These visualizations help interpret the results produced by the optimization models implemented in MATLAB.

## Contents

The figures may include:

* Demand vs. assigned workforce across time slots
* Visualization of employee schedules
* Performance comparison between optimization methods
* Staffing distribution during the planning horizon
